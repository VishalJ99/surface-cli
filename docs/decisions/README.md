# ADRs

This directory stores Architecture Decision Records.

## Format

Each ADR should be a short Markdown file with:

- Title
- Status
- Context
- Decision
- Consequences

## Numbering

Use zero-padded sequential numbers:

- `0001-...`
- `0002-...`
- `0003-...`
